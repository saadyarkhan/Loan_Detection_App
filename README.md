# Loan Approval Predictor

A machine learning web app that predicts whether a loan application is likely to be approved, based on applicant details like income, credit history, and property area. Built as a hands-on project for learning model deployment.

**Live demo:** _add your Streamlit Cloud URL here after deploying_

## How it works

- `train.py` loads the loan applicant dataset, cleans missing values, encodes categorical fields, trains a Logistic Regression classifier, and saves it to `model.pkl`.
- `app.py` is a [Streamlit](https://streamlit.io) app that loads `model.pkl` and lets a user enter applicant details in a form to get a live prediction.

## Project structure

```
.
├── app.py              # Streamlit web app
├── train.py             # Model training script
├── requirements.txt     # Python dependencies
├── train_data.csv       # Training dataset (not included — add your own)
└── model.pkl             # Trained model (generated by train.py)
```

## Running locally

1. Clone this repo:
   ```bash
   git clone https://github.com/<your-username>/<your-repo>.git
   cd <your-repo>
   ```

2. Create a virtual environment and install dependencies:
   ```bash
   python -m venv .venv
   source .venv/bin/activate       # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. Add `train_data.csv` to the project folder (the [Kaggle Loan Prediction dataset](https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset) or your own, with matching columns).

4. Train the model:
   ```bash
   python train.py
   ```
   This creates `model.pkl`.

5. Run the app:
   ```bash
   streamlit run app.py
   ```
   Opens at `http://localhost:8501`.

## Deploying on Streamlit Community Cloud

1. Push this repo to GitHub, including `model.pkl`.
2. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with GitHub.
3. Click **New app**, select this repo and branch, set the main file to `app.py`.
4. Click **Deploy**. Streamlit installs `requirements.txt` automatically and gives you a public URL.
5. Any future push to the branch auto-redeploys the app.

## Model details

- **Algorithm:** Logistic Regression (scikit-learn)
- **Features used:** Gender, Married, Education, Self_Employed, ApplicantIncome, CoapplicantIncome, LoanAmount, Loan_Amount_Term, Credit_History, Property_Area
- **Target:** Loan_Status (Approved / Rejected)
